# Acknowledgments and Provenance

This repository is a new, independent portfolio version of an academic project on Adaptive Role-Play Prompting.

It builds on the original Role-Play Prompting codebase and paper:

- Original repository: https://github.com/NKU-HLT/Role-Play-Prompting
- Original paper: "Better Zero-Shot Reasoning with Role-Play Prompting"
- Original authors: Aobo Kong, Shiwan Zhao, Hao Chen, Qicheng Li, Yong Qin, Ruiqi Sun, and Xin Zhou

The adaptive role-selection experiment, project framing, reports, and README presentation in this repository are part of this portfolio version. The included public datasets and baseline code structure are derived from the original research codebase and related public benchmark resources.

This repository is intended as an educational and portfolio artifact with explicit attribution to the original authors.
